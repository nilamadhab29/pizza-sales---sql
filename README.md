# 🍕 Pizza Sales Analysis Using SQL

## 📌 Project Overview

This project analyzes a Pizza Sales dataset using SQL to extract valuable business insights. The analysis focuses on sales performance, customer ordering behavior, revenue generation, and pizza popularity.

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-selling pizzas.
- Calculate total revenue.
- Understand customer ordering patterns.
- Discover the most popular pizza categories and sizes.

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Relational Database
- Data Analysis

## 📊 Key Business Questions Solved

1. Retrieve the total number of orders placed.
2. Calculate total revenue generated from pizza sales.
3. Identify the highest-priced pizza.
4. Determine the most common pizza size ordered.
5. Find the top 5 most ordered pizza types.
6. Calculate total quantity ordered by pizza category.
7. Analyze order distribution by hour of the day.
8. Determine category-wise pizza distribution.
9. Calculate average pizzas ordered per day.
10. Identify the top 3 pizza types based on revenue.

## 📈 Key Insights
- Total Orders: 21,350
- Total Revenue: $817,860.05
- Highest Priced Pizza: The Greek Pizza ($35.95)
- Most Ordered Size: Large (L)
- Top Revenue Generating Pizza: The Thai Chicken Pizza

## 📂 Database Tables
- Orders
- Order_Details
- Pizzas
- Pizza_Types
